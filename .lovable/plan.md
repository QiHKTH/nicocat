

# 🐱 猫咪体重追踪器

一个极简现代风格的单页应用，用于记录和可视化你家猫咪的体重变化。

---

## 🎨 设计风格

- **配色**：黑白灰极简色调，深灰色文字 + 白色背景 + 浅灰色卡片
- **排版**：大量留白，清晰的层次结构，无缝体验
- **字体**：现代无衬线字体，重点数据使用更大的字号
- **图表**：简洁的折线图，黑色线条配浅灰色填充

---

## 📱 页面布局

### 顶部区域
- 简洁的标题「猫咪体重追踪」
- 猫咪名字显示（可自定义）

### 主要区域 - 体重输入卡片
- 一个输入框，只需输入体重数字
- 点击「记录」按钮后自动获取当前日期保存
- 即时反馈，显示记录成功

### 图表区域 - 体重趋势
- 折线图展示体重变化曲线
- 鼠标悬停显示具体日期和体重
- 支持查看全部数据或最近30天

### 统计卡片区域
- **当前体重**：最新一次记录
- **平均体重**：所有数据的平均值
- **最高/最低**：历史最高和最低体重
- **变化趋势**：相比上次的增减情况

### 数据列表区域
- 可展开的历史记录列表
- 显示日期和对应体重
- 支持删除错误记录

### 底部工具栏
- **导出数据**：一键导出为 CSV 格式
- **清空数据**：带确认的清空功能

---

## 💾 数据存储

- 使用浏览器 LocalStorage 本地存储
- 数据格式简单清晰
- 预填充你提供的 13 条历史数据
- 无需后端，完美兼容 GitHub Pages

---

## ✨ 特色功能

1. **一键记录**：只需输入体重，日期自动获取
2. **趋势可视化**：美观的折线图一目了然
3. **智能统计**：自动计算各项数据指标
4. **CSV 导出**：方便备份或导入其他工具
5. **响应式设计**：手机/平板/电脑都能完美显示

---

## 🚀 技术适配

- 纯前端实现，完全兼容 GitHub Pages 静态托管
- 无需后端服务器，无需数据库
- 代码可直接推送到 GitHub 仓库使用
